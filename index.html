Search & Replace

Quickfix/loclist: cdo is each line (entry), cfdo is each file so use % for whole file

cdo s/\vfoo/bar/g | writereplace per line entry
cfdo %s/\vfoo/bar/g | writereplace per file
  • Omit search string if it's highlighted
  • g match multiple per line, c to confirm
  • \v switches to non-vim regex
  • Wrap in () and use \1, \2 in replace
  • cfdo write write all files
  • cfdo normal undo undo last change to files
